# 買い物価格記録

店舗ごとの商品価格を記録し、商品ごとの最安値を確認するための静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「商品」「店舗」「価格」の順に登録します。
3. 「最安値」「グラフ」で価格を確認します。
4. 必要に応じて「データ」からCSVをエクスポートします。

## GitHub Pagesで公開する場合

このフォルダ内の以下のファイルをリポジトリに配置してください。

- `index.html`
- `style.css`
- `app.js`
- `README.md`

GitHubのリポジトリ設定で Pages を有効にすると、そのまま公開できます。

## 保存先

入力データはログイン不要で、利用中のブラウザの `localStorage` に保存されます。
別の端末や別のブラウザには自動同期されません。バックアップにはCSVエクスポートを使ってください。

## デザイン

白と淡いグレーを基調に、余白を広めにしたミニマルな画面にしています。
カード、タブ、ボタン、入力欄は角丸と控えめな影で整理し、スマホでも価格や入力項目が見やすいように調整しています。
